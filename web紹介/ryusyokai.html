龍翔会 弓道部

URL: https://ryugasakikyudo.vercel.app/

サイト概要

茨城県龍ヶ崎市を拠点に活動する弓道団体「龍翔会」の公式ウェブサイトです。弓道の普及と技術向上を目的とし、初心者向け教室から大会情報、会員向けコンテンツまで、幅広い情報を提供しています。

主なコンテンツ

  • 初心者向け弓道教室の案内
  • 練習スケジュールと年間行事予定
  • 大会・審査会の要項と結果
  • 会員専用ページ（ログイン機能付き）
  • お問い合わせフォーム
  • 活動報告と更新情報

技術スタック

  • コア技術: HTML5, CSS3, JavaScript
  • フレームワーク: Next.js (Vercel ホスティング)
  • スタイリング: CSS Modules
  • 認証: メール認証機能
  • 特徴: レスポンシブデザイン、会員専用エリア

まとめ

このウェブサイトは、弓道愛好家が気軽に情報を取得できることを目的に制作しました。特に、スマートフォンからのアクセスを想定したレスポンシブデザインを採用し、どのデバイスからでも快適にご利用いただけます。会員専用ページでは、メール認証を実装し、セキュアな情報共有を実現しています。

実際のサイトはこちらからご覧いただけます。
